# 🛡️ Z-BRAND PROTECTOR Pro (v2.0)
**Professional Identity Security Framework**

Sistema de grado industrial para la detección de suplantación de identidad y monitoreo de presencia digital. Diseñado para marcas, figuras públicas y departamentos de seguridad.

### ✨ Características de Élite:
- **Multi-Engine Recon:** Integración nativa con Sherlock (escalable a WhatsMyName y Holehe).
- **Automated Reporting:** Generación de informes técnicos en JSON y reportes ejecutivos en TXT.
- **Robustness:** Manejo de dependencias, auditoría de logs y modo de simulación (Dry Run).
- **Security First:** Control de timeouts y manejo de errores críticos.

### 📋 Requisitos:
- Python 3.10+
- Motores de búsqueda instalados (Sherlock, etc.)
